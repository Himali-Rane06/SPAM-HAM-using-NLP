# 📩 Spam-Ham Detection using NLP

## 📌 Overview

This project focuses on building a **Spam vs Ham (Not Spam) classifier** using **Natural Language Processing (NLP)** and **Machine Learning techniques**.
The model analyzes text messages and predicts whether a message is **spam or legitimate (ham)**.

---

## 🎯 Objective

* To classify SMS/email messages as **Spam or Ham**
* To apply NLP techniques for text preprocessing
* To build and evaluate machine learning models for text classification

---

## 📂 Dataset

* The dataset contains labeled messages:

  * **Spam (1)** → Unwanted messages
  * **Ham (0)** → Normal messages
* Typically based on the **SMS Spam Collection dataset**

---

## ⚙️ Technologies Used

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* NLTK (Natural Language Toolkit)

---

## 🔄 Workflow / Methodology

### 1. Data Preprocessing

* Lowercasing text
* Removing punctuation
* Tokenization
* Stopword removal
* Stemming / Lemmatization

### 2. Feature Extraction

* Count Vectorization (Bag of Words)
* TF-IDF Vectorization

👉 Converts text into numerical format for ML models

---

### 3. Model Building

* Logistic Regression
* Naive Bayes
* Random Forest (if used)

---

### 4. Model Evaluation

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

---

## 📊 Results

* Achieved high accuracy in classifying spam messages
* Model successfully identifies important keywords like:

  * "free", "win", "offer"
